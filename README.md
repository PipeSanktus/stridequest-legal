# StrideQuest Legal Pages

Static HTML pages for Google Play Store compliance, hosted on GitHub Pages.

## URLs

- **Privacy Policy**: https://pipesanktus.github.io/stridequest-legal/privacy-policy.html
- **Terms of Service**: https://pipesanktus.github.io/stridequest-legal/terms-of-service.html

## How to publish

1. Create a new GitHub repo named **`stridequest-legal`** at https://github.com/new
2. Copy the contents of this `legal/` folder to the repo root
3. Push to `main` branch
4. Go to **Settings → Pages → Branch: main / folder: / (root)** → Save
5. GitHub Pages will be live at `https://pipesanktus.github.io/stridequest-legal/`

## Updating documents

Edit the HTML files and push to `main`. Changes are live within ~1 minute.

Always update the "Last updated" date at the top of the modified document.
