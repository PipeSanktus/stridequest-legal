ES

Resumen

StrideQuest recopila solo los datos necesarios para autenticar usuarios, sincronizar progreso y operar funciones principales del juego.

Datos que recopilamos

  • Datos de cuenta: correo electronico, identificador de usuario, nombre visible y metodos de autenticacion (Google o email/password).
  • Datos de salud: conteo de pasos leido desde Health Connect (cuando el usuario otorga permiso).
  • Datos tecnicos: token FCM para notificaciones push y registros de errores/uso para diagnostico y mejora del servicio.

Uso de datos

  • Autenticacion y seguridad de sesion.
  • Sincronizacion de progreso, eventos y misiones.
  • Entrega de notificaciones y operacion general de la app.

Comparticion

No vendemos datos personales. Usamos proveedores tecnicos para operar la app (por ejemplo, servicios de Firebase de Google).

Eliminacion de cuenta

Puedes solicitar la eliminacion de tu cuenta y datos asociados en Account Deletion.

EN

Summary

StrideQuest collects only the data required to authenticate users, sync progress, and operate core gameplay features.

Data we collect

  • Account data: email, user identifier, display name, and authentication method (Google or email/password).
  • Health data: step count read from Health Connect (when permission is granted).
  • Technical data: FCM token for push notifications and diagnostics/usage logs for service quality.

How data is used

  • Authentication and session security.
  • Progress sync, events, and missions.
  • Push notifications and overall app operations.

Data sharing

We do not sell personal data. We use technical providers to run the app (for example, Google Firebase services).

Account deletion

You can request account and associated data deletion at Account Deletion.
